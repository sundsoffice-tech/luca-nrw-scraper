**Features:**
- Status badges (Aktiv/Inaktiv)
- Service descriptions
- Direct links to configuration pages
- Information box with admin link

## Color Scheme & Design

All settings pages use the existing TELIS CRM dark theme:

- **Background**: `#0f172a` (dark-900)
- **Cards**: `#1e293b` (dark-800)
- **Borders**: `#334155` (dark-700)
- **Primary**: `#06b6d4` (cyan-500)
- **Text**: `#f1f5f9` (light)
- **Text Secondary**: `#94a3b8` (gray-400)

## Responsive Design

All pages are fully responsive:
- **Desktop (lg)**: 3-column grid for tiles
- **Tablet (md)**: 2-column grid
- **Mobile**: Single column
- Sidebar collapses to overlay menu on mobile

## Interaction States

- **Hover**: Border changes to primary color, subtle shadow
- **Active**: Background highlight on current page
- **Focus**: Ring around focused inputs
- **Success**: Green notification banner
- **Error**: Red notification banner

## Accessibility

- Semantic HTML structure
- Clear labels for all form elements
- Keyboard navigation support
- High contrast text
- Icon + text for all actions
